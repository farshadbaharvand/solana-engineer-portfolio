# معرفی Ankr: زیرساخت غیرمتمرکز برای Web3

یک پلتفرم پیشرو در حوزه زیرساخت‌های بلاک‌چین (Blockchain Infrastructure) است که با هدف تسهیل دسترسی توسعه‌دهندگان و شرکت‌ها به شبکه‌های غیرمتمرکز طراحی شده است. این پلتفرم با ارائه ابزارهای قدرتمند، امکان تعامل سریع و ایمن با چندین بلاک‌چین را فراهم می‌کند

---


## خدمات اصلی **Ankr**

پلتفرم **Ankr** مجموعه‌ای گسترده از سرویس‌ها را برای اکوسیستم **Web3** ارائه می‌دهد که مهم‌ترین آن‌ها عبارت‌اند از:

---

### ✅ **RPC Service**  
**(سرویس فراخوانی رویه از راه دور)**  
ارائه دسترسی سریع، پایدار و مقیاس‌پذیر به **Blockchain Nodes** در شبکه‌های مختلف بلاک‌چین.

---

### ✅ **Staking Solutions**  
**(راهکارهای استیکینگ)**  
امکان مشارکت در فرآیند **Staking** توکن‌ها و دریافت پاداش، بدون نیاز به مدیریت پیچیده زیرساخت یا سخت‌افزار.

---

### ✅ **AppChains**  
**(زنجیره‌های اختصاصی برنامه)**  
قابلیت راه‌اندازی بلاک‌چین‌های شخصی‌سازی‌شده برای پروژه‌های مقیاس‌پذیر و کاربردهای سازمانی.

---

### ✅ **Developer Tools**  
**(ابزارهای توسعه‌دهنده)**  
مجموعه‌ای از ابزارها برای تسهیل در استقرار، مدیریت و تعامل با **Smart Contracts**.


---

## چرا توسعه‌دهندگان از Ankr استفاده می‌کنند؟

استفاده از **Ankr** به دلایل فنی زیر برای تیم‌های توسعه اهمیت دارد:

### ۱. مقیاس‌پذیری (Scalability)
**Ankr** با استفاده از یک شبکه جهانی از گره‌ها (Nodes)، تأخیر (Latency) را به حداقل می‌رساند و اطمینان حاصل می‌کند که درخواست‌های برنامه (DApps) به سرعت پردازش می‌شوند.

### ۲. پشتیبانی از چند زنجیره (Multi-chain Support)
این پلتفرم از شبکه‌های بلاک‌چینی متعددی پشتیبانی می‌کند و به توسعه‌دهندگان اجازه می‌دهد بدون تغییر زیرساخت، بین شبکه‌های مختلف جابجا شوند.

### ۳. سهولت در راه‌اندازی
با استفاده از **Ankr**، فرآیند راه‌اندازی گره‌ها که در حالت عادی بسیار زمان‌بر و هزینه‌بر است، به حداقل می‌رسد.

---

## شروع کار با Ankr

برای شروع استفاده از سرویس‌های **Ankr**، مراحل کلی زیر طی می‌شود:

1. **ایجاد حساب کاربری:** مراجعه به وب‌سایت رسمی و ثبت‌نام در پنل کاربری.
2. **دریافت API Endpoint:** دریافت کلید دسترسی برای اتصال برنامه خود به شبکه بلاک‌چین مورد نظر.
3. **اتصال به شبکه:** استفاده از **Endpoint** دریافتی در کد پروژه.

به عنوان مثال، برای اتصال به یک شبکه اتریوم (Ethereum) از طریق **Ankr**، می‌توانید از قطعه کد زیر در محیط **Node.js** استفاده کنید:

```javascript
const Web3 = require('web3');

// استفاده از RPC Endpoint ارائه شده توسط Ankr
const rpcURL = "https://rpc.ankr.com/eth";
const web3 = new Web3(rpcURL);

web3.eth.getBlockNumber().then((blockNumber) => {
    console.log("Current block number:", blockNumber);
});
```
